# USC Social Circle Map

Vue 3 + Mapbox GL JS prototype with multiple synthetic USC students.

## Setup

1. Create a `.env.local` file in the project root:

```bash
VITE_MAPBOX_TOKEN=your_mapbox_token_here
```

2. Install dependencies:

```bash
npm install
```

3. Run the app:

```bash
npm run dev
```

## What it does

- Shows multiple students on a Mapbox map centered on USC
- Lets you click a student marker to switch the selected profile
- Updates the sidebar stats, trend chart, places, and people for that student
- Uses synthetic data only
